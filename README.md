# 🔮 命运工坊 - 综合占卜与命理应用

一个精美的综合占卜与命理Web应用，集成了多种传统和现代占卜方式，包括塔罗牌、八字算命、奇门遁甲、风水罗盘等功能。

## ✨ 功能特性

### 🔮 塔罗占卜
- 🎴 **完整牌组**：包含78张塔罗牌的完整信息（22张大阿卡纳 + 56张小阿卡纳）
- 🔮 **单张抽牌**：随机抽取一张塔罗牌
- 🌟 **三牌占卜**：一次性抽取三张牌进行占卜
- 🔄 **正逆位切换**：每张牌都可以查看正位和逆位的含义
- 📖 **详细解释**：每张牌都包含详细的牌意和描述
- 📚 **历史记录**：保存占卜历史，支持导出和分享

### ✨ 智能取名
- 基于五行、八字等传统命理学的智能取名服务
- 提供个性化的名字建议

### ♈ 星座运势
- 十二星座每日运势查询
- 详细的星座性格分析和建议

### 📅 今日黄历
- 传统农历历法查询
- 宜忌事项、吉凶时辰
- 传统节日和节气信息

### 🙏 赛博积德
- 数字化的积德体验
- 木鱼、放生、上香、祈福等功能
- 功德统计和记录

### ☯ 八字算命
- 基于出生日期计算八字
- 五行分析、命理解读
- 传统命理学应用

### 🎋 抽签求签
- 传统抽签功能
- 签文解读和建议

### 💭 梦境解析
- 梦境记录和解析
- 潜意识探索

### 🧭 风水罗盘
- 方位吉凶分析
- 风水建议

### ⏰ 择日吉时
- 良辰吉日查询
- 重要事项择日建议

### 🔢 数字能量
- 数字命理学分析
- 数字组合能量解读
- 个性化数字建议

### 🎨 每日幸运色
- 基于日期和个人信息的幸运色推荐
- 支持农历/阳历转换
- 根据生肖、星座、八字五行推荐
- 详细的颜色能量解读

### ⚡ 奇门遁甲
- 传统预测术，分析吉凶方位和时间

## 🎯 使用说明

### 塔罗占卜
1. **抽取单张牌**：点击"抽取一张牌"按钮，随机抽取一张塔罗牌
2. **三牌占卜**：点击"三牌占卜"按钮，一次性抽取三张牌
3. **查看牌意**：抽取后可以查看牌的正位和逆位含义
4. **翻转卡片**：点击卡片或"转为逆位/正位"按钮来切换正逆位
5. **选择已抽牌**：在已抽取的牌列表中点击任意一张牌来查看详情
6. **重新开始**：点击"重新开始"按钮清空所有已抽取的牌

### 每日幸运色
1. 选择查询日期
2. 可选择输入个人信息（出生日期、生肖、星座）获得个性化推荐
3. 支持农历/阳历转换
4. 查看幸运色详细解读和使用建议

### 数字能量
1. 输入数字进行分析
2. 查看数字组合的能量解读
3. 了解每个数字的详细含义

### 奇门遁甲
1. 选择时间和方位
2. 查看九宫格盘面分析
3. 了解各宫位的八门、九星、八神
4. 获取吉凶方位建议

### 其他功能
- 通过3D旋转轮播选择不同功能
- 每个功能都有详细的说明和指导
- 支持移动端和桌面端访问

## 🎨 牌组说明

### 大阿卡纳 (Major Arcana)
22张牌，代表人生的重要阶段和转折点，包括：
- 愚者、魔术师、女祭司、皇后、皇帝、教皇
- 恋人、战车、力量、隐者、命运之轮、正义
- 倒吊人、死神、节制、恶魔、塔、星星
- 月亮、太阳、审判、世界

### 小阿卡纳 (Minor Arcana)
56张牌，分为四个花色：
- **权杖 (Wands)**：代表行动、创造力和热情
- **圣杯 (Cups)**：代表情感、直觉和关系
- **宝剑 (Swords)**：代表思想、沟通和冲突
- **星币 (Pentacles)**：代表物质、工作和资源

每个花色包含14张牌：Ace到10，以及侍从、骑士、皇后、国王。

- 本应用仅供娱乐参考，占卜结果不应作为重要决策的唯一依据
